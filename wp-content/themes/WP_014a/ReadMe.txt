Soccer Dog Free Pets Niche Wordpress Theme

Demo: http://mag1.31f.info/?wptheme=WP_014a

Features of this theme:
* Tested on latest Wordpress versions 2.8.x;
* 3 columns;
* Featured Posts section (uses featured content gallery plugin);
* Featured video section (can be cofigured by editing featured-video.inc);
* Fixed width;
* Widget ready;
* Gravatar ready;
* Adsense ready (can be configured by 300x250.inc, 468x60.inc);
* Site-wide 468 x 60 Header banner Ads. (can be configured by 468x60.inc);
* Inner page 468 x 60 Adsense banner Ads. (can be configured by 468x60.inc);
* Site-wide 300 x 250 Sidebar Adsense banner Ads. (can be configured by 300x250.inc);
* Site-wide 125 x 125 Sidebar banner ads; (uses wp125 plugin);
* Threaded Comments;
* WP-Pagenavi integrated; (wp-pagenavi plugin required)
* Cross browser compatible with IE, FF, Opera, Flock & Safari;
* Theme support/customization via our website;

*** Pre-requisites to use this theme
Theme requires following plugins. If you are not installing these plugins you will not enjoy all the features of the theme. So, yes go ahead and download the following plugins.

1. wp-pagenavi plugin
2. wp 125 plugin
3. Featured-content-gallery

Download all required plugins from http://www.wpspider.com/downloads/pets-wordpress-themes-plugins.zip

Installation instructions and other details can be found @ http://www.wpspider.com/pets-wordpress-themes-installation-guide/
